# Brukerveiledning

## Lag din egen fork av mal repoet

Gå til https://github.com/Arkitektum/produktspesifikasjon_mal

Trykk på fork oppe til høyre, velg hvor du skal legge det (på deg selv eller en organisasjon f.eks.), og trykk create fork nede til høyre.
